PhiSat-2 Quickstart

Minimal steps to get a first run working.

DocsGetting started

1) Install

git clone https://github.com/clementgilli/phisat2
cd phisat2
make install

Requirements: Python 3.13 + uv + make.

2) Verify runtime

make check
make list-models
make list-dataloaders

3) Smoke test

Runs one fast synthetic batch with no local Earth data.

make smoke

4) First training command

make fast-dev-run \
  TASK=segmentation \
  DATASET=clouds \
  MODEL=phisat2_geoaware \
  DATALOADER=synthetic \
  SEEDS=0 \
  ROOT_DIR=. \
  OUTPUT_DIR=runs/smoke \
  EPOCHS=1 \
  BATCH_SIZE=2 \
  NUM_WORKERS=0 \
  ACCELERATOR=cpu \
  DEVICES=1 \
  PRETRAINED=false \
  LR=0.0001

Equivalent command exists under README.md via uv run ... -m phisat2.cli.train fit ....
