Inference notes

How to run a model on new inputs.

DocsInference

Checkpoint workflow

  1. Instantiate the model with build_model(name, spec, pretrained=False).
  2. Load checkpoint weights from runs/.../checkpoints/.
  3. Keep inputs shaped as [B, 8, H, W] and normalized like training.

Output shape

  • Segmentation and pixel regression: [B, C, H, W].
  • Classification: [B, 31].
  • Global regression: [B, 4].

Note

There is no dedicated inference CLI yet. Use a short Python script with the same model construction path as training.
