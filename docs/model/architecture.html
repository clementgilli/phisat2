Model registry

phisat2/models/registry.py is the source of truth:

  • phisat2_geoaware → shared decoder
  • terramind_* → shared decoder
  • prithvi_* → shared decoder
  • dofa_* → shared decoder
  • seco_* → shared decoder
  • ssl4eos12_* → shared decoder
  • satlas_* → shared decoder
  • myriad2_full_unet → full-structure decoder

Shared-decoder models

  • Dispatcher: build_model(name, spec, pretrained) wraps encoder with SharedDecoderModel.
  • phisat2_geoaware uses PhiSat2GeoAwareEncoder.
  • TerraTorch-backed names use TerraTorchBackboneEncoder.

SharedDecoderModel flow: encoder output → adapter → task head.

  • Spatial tasks (segmentation, pixel_regression): SharedUNetDecoder.
  • Global tasks (classification, global_regression): GlobalPoolingHead.

Feature adaptation

FeaturePyramidAdapter makes heterogeneous encoder outputs compatible:

  1. select 4 feature maps from encoder output
  2. project each to channels (32, 64, 128, 256) via LazyConv2d
  3. resize spatial scales to expected pyramid levels

So raw feature maps can differ by encoder, but decoder input contract stays identical.

Exception model

myriad2_full_unet is not wrapped.

  • Built as a complete U-Net with encoder + decoder.
  • Supports only segmentation and pixel regression.
  • Uses 3 down/upsampling stages with skip concatenation.
