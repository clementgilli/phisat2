Common errors and quick fixes.

Common failures

  1. Invalid task/dataset: check phisat2/tasks/specs.py and selected dataloader support.
  2. TerraTorch import error: install group dependencies and retry make install.
  3. Loader path error: ensure ROOT_DIR points to existing .zarr or .h5 dataset tree.
  4. Shape mismatch: this usually appears in loss/logits size checks for segmentation targets.
  5. CUDA/DDP mismatch: use one device first; then enable AUTO_DDP on multi-GPU.

Fast debug flow

make check
make list-models
make smoke

If smoke passes, the issue is typically data path or task-model compatibility.
