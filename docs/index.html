PhiSat-2

A Python 3.13 training package for PhiSat-2 experiments using Lightning, TerraTorch-compatible backbones, and Makefile-driven runs.

Quickstart

Install dependencies with uv and the project Makefile.

git clone https://github.com/clementgilli/phisat2
cd phisat2
make install

Requirements: Python 3.13, uv, and make.

How to Run

Use the Makefile as the public interface. Start with a lightweight import check, then inspect available runtime options.

make check
make list-models
make list-dataloaders

Run a one-batch CPU smoke test with synthetic data:

make smoke

Run the test suite:

make test

Training

Experiments are configured with Make variables. The default output directory is runs.

make train TASK=segmentation DATASET=lulc MODEL=terramind_v1_tiny DATALOADER=zarr_downstream SEEDS="13 42 100" EPOCHS=50
make train TASK=classification DATASET=climate MODEL=prithvi_eo_v1_100 DATALOADER=h5_pairs SEED=42 EPOCHS=30
make train TASK=pixel_regression DATASET=reconstruction MODEL=phisat2_geoaware DATALOADER=h5_pairs SEED=7 BATCH_SIZE=16

The equivalent Python entrypoint is available after install:

uv run --python 3.13 python -m phisat2.cli.train fit --task segmentation --dataset clouds --model phisat2_geoaware --dataloader synthetic --seeds 0 --root-dir . --output-dir runs/smoke --max-epochs 1 --batch-size 2 --num-workers 0 --accelerator cpu --devices 1 --no-pretrained --fast-dev-run

Dataset names are task-specific. The CLI reports an error if a dataset is not valid for the selected task.

Data Loaders

  • zarr_downstream: Zarr downstream segmentation datasets with image/mask batches.
  • h5_pairs: Paired PhiSat-2/Sentinel HDF5 data for climate, geolocation, and reconstruction.
  • synthetic: Small random batches used by smoke tests and CI.

Use ROOT_DIR=/path/to/data for real datasets. The smoke test uses DATALOADER=synthetic and does not require local data.

Notes

Files under pretrain/weights are reference artifacts and are not used to initialize training.

myriad2_full_unet preserves the inspected full-U-Net topology as a benchmark exception and only supports spatial tasks.
