Repository map (short)

  • phisat2/: active package for training + CLI.
  • src/: legacy/adaptation scripts and alternate training paths.
  • tests/: registry and model shape checks.

Quick run order

  1. make install
  2. make list-models and make list-dataloaders
  3. make smoke
