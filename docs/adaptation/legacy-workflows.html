Legacy adaptation workflows

Scripts in src/adaptation/ and when to use them.

DocsLegacy workflows

Scripts

  • train_distillation_student.py: SIM to REAL encoder distillation.
  • train_distillation_teacher.py: teacher/student feature matching.
  • train_downstream.py: frozen encoder plus task decoder training.
  • train_pretext.py: climate, geoloc, and reconstruction probes.

Boundary

These scripts are separate from the public CLI and are mainly for research-style experiments.
